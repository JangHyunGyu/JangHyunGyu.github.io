ArcherLab 개요

Archer, Lab, .dev. 작업 범위와 방법을 정의합니다.

Archer

집중과 정밀. 문제를 정의하고 해결합니다.

Lab

실험과 탐구. 가설을 세우고 결과를 기록합니다.

.dev

코드 중심. 브라우저와 에디터로 프로토타입을 만듭니다.

Motivation

익숙함 속의 불편을 포착하고 체계적으로 개선합니다.

Philosophy · Archer's Notes

작업 기준을 간단히 정리합니다.

기술은 수단, 기준은 사용자.

인터페이스를 기능, 성능, 접근성 기준으로 설계합니다.

ArcherLab은 실험 기록이다.

중간 산출물과 실패 로그도 남깁니다. 재시도에 활용합니다.

단편 지식을 구조화해 기록한다.

산발적인 경험과 지식을 맥락으로 묶어 재활용 가능한 기록으로 남깁니다.

최근 실험 목록

최근 개인 시간에 진행한 실험 목록입니다. 월 단위로 갱신합니다. 대부분 탐색 단계입니다.

EchoTrace

오픈소스 LLM을 미니 PC에 올리고, 내 음성을 즉시 코드 스니펫으로 바꿔주는 음성-코드 인터페이스.

NightShift Deck

키캡 아래에 칩LED를 삽입한 커스텀 매크로 패드. 음향 효과와 빠른 매핑으로 라이브 코딩을 지원합니다.

404 Observatory

실시간 트래픽 로그를 시각 패턴으로 변환합니다.

Capsule Lab

익명 큐레이션 서버. 짧은 초대 링크로 피드백을 수집합니다.

손에 익은 도구들과 새로 배우는 것들

프런트엔드, 사운드, 전자공학을 필요에 맞게 조합합니다. 새 기술은 작은 단위로 실험합니다.

Symmetric Stack

TypeScript, Astro, SvelteKit, Three.js 조합으로 인터랙티브 웹을 빠르게 스케치합니다.

Physical Playground

Raspberry Pi 5, ESP32, 심박 센서를 결합해 실시간 감정 반응을 캡처합니다.

Sonic Rituals

Ableton Live와 Max for Live로 모듈러 신스와 코드를 연결해 데이터 기반 사운드를 만듭니다.
